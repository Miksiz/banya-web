🧖 Нажмите, чтобы войти в парилку

💈 Управление

WASD / Стрелки — движение

Мышь / Свайп — осмотр

C — выключить/включить режим взаимодействия

J — пересоздать мир

ESC — выйти из парилки

В режиме взаимодействия (по-умолчанию включен):

ЛКМ — взять/положить объект

Колёсико — отдалить/приблизить взятый объект

ПКМ + движение — вращение взятого объекта

Ср.кн.мыши — удалить объект

X/Z — переключить оси вращения (чтобы при вращении вращать только во включенном направлении)

N — переключение режима отладки физических коллайдеров

M — записать в лог текущую позицию камеры

B — записать в лог информацию о выбранном объекте

🌡️ 85°C
