💈 Парилка

WASD — передвижение

Мышь — осмотр

C — режим выделения

ESC — выйти

В режиме выделения:

ЛКМ — взять объект

ПКМ — повернуть взятый объект (зажать)

СКМ — удалить объект

Для поворота объектов:

X — переключение горизонтального вращения

Z — переключение вертикального вращения
